Тестовый сценарий 1.1.1 Переход на страницу "Все товары"
Предусловия:
- Открыта страница http://selenium1py.pythonanywhere.com/ru/#;
- На странице есть кнопка с текстом "Все товары"
- Известно количество всех товаров в БД
Шаги:
- Шаг 1
Щелкнуть по кнопке с текстом "Все товары"
- проверка
	1.На сервер пошел запрос get с адресом страницы http://selenium1py.pythonanywhere.com/ru/catalogue/
	2.Проверка шаблона
	3.Строка навигации
	4.Количество результатов
	5.Количесвто отображаемых продуктов
- ожидаемый результат
	1.Вместо исходной страницы загрузилась новая с url 	http://selenium1py.pythonanywhere.com/ru/catalogue/
	2.Контент страницы включает все элементы шаблона страницы "Все товары"
	3.В строке навигации указано нахождение на странице "Начало / Все товары"
	4.Количество результатов соответствует существующему.
 	5.На странице отображается результат с элементами с 1 по 20, всего 20 элементов. 

Тестовый сценарий 1.2.1 Выбор из БД существующего товара.
Предусловия: 
- Открыта страница http://selenium1py.pythonanywhere.com/ru/catalogue/;
- Контент страницы соответствует шаблону;
- На открытой странице есть хотя бы один товар;
Шаги:
- Шаг 1
	Скопировать со страницы текст ссылки первого элемента в строку поиска.

	1.На сервер ушел запрос get с содержимым текстом ссыки.
	2.Верстка страницы
	3.Название отображаемого продукта, количество отображаемых продуктов
	4.Количество найденных результатов
- ожидаемый результат
	1.На сервер ушел запрос get с содержимым текстом ссыки.
	2.Верстка новой странички соответствует шаблону
	3.На страничке один найденный продукт, текст ссылки равен тексту поисковой строки
	4.Количество найденных результатов =1

Тестовый сценарий 2.2.1 Переход на страницу подробностей о товаре
Предусловия:
- Открыта страница http://selenium1py.pythonanywhere.com/ru/catalogue/;
- На странице есть хотя бы один продукт.
Шаги:
- Шаг 1
Щелкнуть по ссылке с текстом наименования первого продукта  в области отображения продуктов
- проверка
	1.На сервер пошел запрос get с адресом страницы http://selenium1py.pythonanywhere.com/ru/catalogue/"текст ссылки на первый продукт"+номер/ (слова текста разделены дефисом)
	2.Проверка шаблона
	3.Название продукта
	4. Ссылка на картинку
	5. Стоимость
	6.Количество экземпляров
	7.Информация о товаре
- ожидаемый результат
	1.Вместо исходной страницы загрузилась новая с url 	http://selenium1py.pythonanywhere.com/ru/catalogue/"текст ссылки на первый продукт"+номер/ (слова текста разделены дефисом)
	2.Контент страницы включает все элементы шаблона страницы "Подробности о продукте"
	3.Текст наименования продукта равен тексту исходной ссылки.
	4.ссылка Пикча продукта не изменилась.
	5.значение стоимости продукта больше нуля
	6.Значение количества экземпляров больше равно нулю.
	7.Таблица "Ифнормация о товаре", поле значений  
		- "Цена (с НДС)" равна надписи цены в заголовке товара;
		- "Доступность" включает цифру из надписи доступности на складе в заголовке товара;
		- "Количество отзывов" равно количеству отзывов в разделе страницы "Отзывы Клиентов"
